# Types de pages – Portail S10

## Objectif

Normaliser la structure des pages afin de garantir :
- homogénéité
- rapidité de production
- compréhension immédiate

Aucune page ne doit exister hors de ces modèles.

---

## Page institutionnelle

### Usage
- Présentation de l’entité
- Missions
- Organisation
- Valeurs

### Sections typiques
- Hero institutionnel
- Texte structuré
- Blocs informatifs
- Appels à l’action sobres

---

## Page service

### Usage
- Description d’un service rendu
- Information pratique

### Sections typiques
- Titre clair
- Description synthétique
- Détails structurés
- Contacts ou liens utiles

---

## Page contenu long

### Usage
- Informations détaillées
- Pages réglementaires
- Dossiers explicatifs

### Sections typiques
- Introduction
- Sections thématiques
- Conclusion ou ressources

---

## Page actualité

### Usage
- Communication ponctuelle
- Information datée

### Sections typiques
- Titre
- Date
- Contenu principal
- Liens associés

---

## Règles générales

- Chaque page correspond à un type défini
- Les sections doivent être cohérentes avec le type
- Toute variation doit rester marginale
- Aucun mélange de logiques entre types
