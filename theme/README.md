# Thème Portail S10

Ce thème WordPress sert de base sur mesure pour les portails S10.

## Objectifs

- Industrialiser la génération de sites institutionnels.
- Offrir une base réutilisable pour les services d'incendie et de secours.

## Fichiers principaux

- `style.css` : métadonnées du thème + styles de base.
- `functions.php` : activation des fonctionnalités et assets.
- `index.php` : gabarit principal.
- `header.php` / `footer.php` : structure HTML globale.
